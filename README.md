## ��� ¿Qué es V-List?

**V-List** es una aplicación web moderna de gestión de tareas construida con **Vue.js 3**. Permite a los usuarios crear, editar, eliminar y organizar sus tareas de forma rápida e intuitiva.

---

## ✨ Características

✅ **Autenticación** - Login y registro de usuarios  
✅ **Crear Tareas** - Agrega nuevas tareas con descripción  
✅ **Editar Tareas** - Modifica tus tareas cuando sea necesario  
✅ **Eliminar Tareas** - Borra tareas completadas o innecesarias  
✅ **Marcar como Completado** - Marca tareas como hechas  
✅ **Filtrar Tareas** - Visualiza todas, pendientes o completadas  
✅ **Datos Persistentes** - Se guardan en localStorage  
✅ **Interfaz Responsiva** - Funciona en móvil, tablet y desktop
